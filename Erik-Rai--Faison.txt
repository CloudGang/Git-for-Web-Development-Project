1. What is the difference between an inline element and a block element?
  Inline element does not cause a line break. | A block element is any element that starts a new line, like a paragraph, and uses the full width of the page or container.
  
2. What happens when an element is positioned absolutely? 
  When an element is positioned absolutely, its position becomes relative to its containg block.
